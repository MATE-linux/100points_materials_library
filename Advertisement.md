# Объявление: Начинается набор в корреспонденты сайта  
1. Требования:  
  - [ ] Быть записанным на полуторагодовой курс в 100 бальном по любому предмету
  - [ ] Быть готовым задерживаться на 10-15 минут после урока для того, чтобы скачать и опубликрвать конспекты
  - [ ] Иметь Telegram или другой способ для связи
2. Технические требования:  
  - [ ] Иметь ПК с доступом в интернет
  - [ ] [Зарегистрироваться на github](https://github.com)
  - [ ] [Установить Obsidian](https://obsidian.md/) для редактирования `.md` файлов, из которых генерируется сайт 
Технический процесс:    
    В первый раз:  
        1. Запросить у разработчика исходники сайта  
        2. Открыть папку `content` в Obsidian, создать файлы согласно структуре:    
``` text
            project/
            ├── content/
            │   └── Math 10 class/ (папка предмета)
                    └── All lessons 
            │       └── ... (файлы конспектов согласно шаблону из templates/Lesson Template.md)
                ├── resourses/ 
                    └── Math_10_class/ (ресурсы предмета - например pdf)
                │       └── Block_1/ (ресурсы за блок 1)
                │           └── теория_треугольники_зантие_2.pdf
                └── templates/
                |   └── Lesson Template.md
```

3. В Obsidian на крайней левой панели нажать кнопку со стрелочками с подписью `Open Git source control`, справа появится панель, на ней сверху нажать кнопку с кружком со стрелкой вверх (подпись `Commit-and-sync`)  
4. Через 5 минут изменения появятся на сайте  

В последующие разы:  
    1. Сохранить `конспект.pdf` в папку `resourses`  
    2. Создать файл конспекта в папке предмета  
    3. Добавить ссылку на конспект в файл `All lessons` в папке вашего предмета
    4. В Obsidian на крайней левой панели нажать кнопку со стрелочками с подписью `Open Git source control`, справа появится панель, на ней сверху нажать кнопку с кружком со стрелкой вверх (подпись `Commit-and-sync`)  
    5. Через 5 минут изменения появятся на сайте
Откликнуться на предложение и запросить исходники можно в комментариях под постом: https://t.me/points_library_update_bot/14
